# Dota 2 Skins Manager

This project provides a set of scripts to manage, install, and update Dota 2 skins and mods.

## Features

- **InstallModsDota.bat**: Automatically installs Dota 2 mods.
- **FixMatchMaking.bat**: Fixes matchmaking issues potentially caused by mods.
- **replace_dota_fonts.bat**: Replaces Dota 2 fonts for a customized experience.

## Usage

1. Clone the repository.
2. Ensure `DotaPath.txt` contains the correct path to your Dota 2 installation.
3. Run the desired `.bat` file as Administrator if required.
